-ouvrir le terminal
-tapez les informations de connexion exemple : ssh -p 65002 u212330119@89.117.9.183
-tapez le mot de passe pour se connecter par ssh: kdkkqqdjq
-spécifier un chemin pour l'installation de composer :
Etape:

>cd txbest.online/public_html/obp/ : ce déplacer dans le dossier txbest.online/public_html/obp/
